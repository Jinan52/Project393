Digger Game Demo

Author:
Jinan Assi

Description:
This project is a demo version of the classic Digger game.
The player moves through the map, digs tunnels, collects gold,
avoids enemies, and can shoot enemies using the space key.

Features:
- Player movement using arrow keys
- Digging mechanic
- Initial tunnels similar to the original Digger game
- Enemies spawn from the upper-right side
- Shooting using Space
- Falling bonus bags
- Bonus bags can kill the player or enemies when falling
- Gold collection
- Heart bonus for extra life
- Score and lives displayed in the window title
- Win and game-over conditions

Controls:
Arrow keys - Move player
Space - Shoot

Technologies:
C++
SDL3
SDL_image

Project Structure:
main.cpp - Entry point
Game.h / Game.cpp - Main game logic
Utils.h / Utils.cpp - Utility functions
res/ - Game images and assets
